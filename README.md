# 3D & Real-Time Development Portfolio

This repository presents a selection of real-time 3D and virtual reality projects developed using Unity, Unreal Engine, and Autodesk 3ds Max.

The projects focus on immersive interaction, real-time visualization, and engineering-oriented 3D workflows.

## Unity Projects

### Classroom Collaborative (VR)
Multi-user VR classroom enabling collaborative interaction in a shared virtual environment.
- Technologies: Unity, Netcode for GameObjects, XR Interaction Toolkit, Vivox  
- Focus: VR collaboration, networking, real-time interaction  

 `unity/classRoomCollaborative`

---

### Parkinson VR Simulation
Virtual reality experience designed to raise awareness about Parkinson’s disease by simulating motor impairments during everyday tasks.
- Technologies: Unity, VR, Blender  
- Context: XR Franco-German Hackathon – Paris 2025  

 `unity/Parkinson`
